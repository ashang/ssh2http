# ssh2http

A restricted login shell for Git-only SSH access,
which delegates git pull/push requests to a git http backend.

## installation

    gem install ssh2http

## usage

    'chsh' -s $(command -v ssh2http) <user>
    'git clone' <user>`@localhost:/path/to/repo.git`
    'ssh' <user>`@localhost`
